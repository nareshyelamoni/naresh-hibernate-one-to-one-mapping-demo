# Hibernate One-to-One Mapping Demo

## Overview

This project demonstrates One-to-One relationship mapping using Hibernate ORM and JPA annotations.

## Technologies Used

* Java
* Hibernate ORM
* JPA (Jakarta Persistence)
* MySQL
* Maven

## Project Structure

* Question Entity
* Answer Entity
* OneToOneMappingDemo

## Relationship

One Question is associated with exactly one Answer.

### Example

Question:

* What is Hibernate?

Answer:

* Hibernate is an ORM Framework.

## Features

* One-to-One Mapping
* Hibernate Configuration using hibernate.properties
* Cascade Operations
* Transaction Management
* MySQL Database Integration

## Learning Outcomes

* Understanding One-to-One Relationships
* Entity Mapping using JPA Annotations
* Data Persistence using Hibernate
